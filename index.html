What is Open Source Software (OSS)?

Software distributed under a license that implies source code sharing.

While proprietary software usually hide source code from user.

Users are free to read the code, modify it and redistribute.

That does not mean that OSS is free of charges.

What is Open Research?

Reproducible and reviewable Research.

Open data: accessible and searchable

Open tools: OS software.

Open access to papers.

That does not mean that Oper Research is free of charges.
